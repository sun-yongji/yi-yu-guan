# 贡献指南 · 易宇观

感谢你对「易宇观」理论体系的关注！本文档说明如何参与核心理论的研究与完善。

## 谁可以参与

- 数学/物理/天文学研究者
- 易学/国学/东方哲学研究者
- 系统科学/复杂性科学研究者
- 技术文档撰写者与翻译志愿者

## 贡献类型

### 理论研究
- 新的理论推导与数学证明
- 模型参数优化与验证
- 跨学科交叉研究

### 文档建设
- 现有文档的校对与修订
- 新研究方向的综述撰写
- 中英文翻译（参考 `README.en.md`）

### 案例研究
- 历史事件的理论回溯验证
- 新现象的理论解释
- 实验设计与验证方案

## 贡献流程

1. Fork 本仓库
2. 创建分支：`git checkout -b theory/your-topic`
3. 提交修改：`git commit -m "theory: add xxx"`
4. 推送 → 提交 Pull Request

## Commit 规范

```
<type>: <description>
```

| type | 用途 |
|------|------|
| theory | 理论更新 |
| docs | 文档修订 |
| translate | 翻译 |
| fix | 修正错误 |

## 文档规范

- 使用 Markdown 格式
- 理论推导需注明假设与前提
- 数学公式使用 LaTeX 语法
- 引用数据需注明来源

## 社区精神

- **严谨求实**：理论需可验证、可复现
- **开放包容**：尊重不同学术背景
- **长期主义**：共建可持续的研究体系
