# 易宇研究中心 · 私有知识库

## 🏛️ 机构定位

易宇研究中心是专注于太极场理论跨学科研究的学术机构，涵盖宇宙学、量子物理、意识科学及传统易学。

**研究方向**：
- 太极场模型的数学构建与科学验证
- 暗物质/暗能量的太极解读
- 意识本质的相干场模型
- 东方数理在量子计算中的应用
- 文明演化律与地磁-八卦耦合

## 📝 内部管理规范

### 文档版本规范

| 版本 | 含义 |
|------|------|
| v1.x | 初稿/草稿 |
| v2.x | 定稿/公开发布版本 |
| v3.x | 重大修订 |

### 评审流程

1. 作者完成初稿 → 提交 PR
2. 至少 2 位核心研究员评审
3. 评审通过 → 合并 → 版本号 +0.1
4. 定稿版本（v2.0+）需全体核心研究员确认

### 对外发布流程

1. 私有库定稿（v2.0）
2. 提炼简要版（500-800字）
3. 提交到公开仓库 `yi-yu-community/yi-yu-core`
4. 公开仓库版本号与私有库对应
