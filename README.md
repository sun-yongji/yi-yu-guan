## 项目简介

易宇观（Yi-Yu-Guan）是「易宇本源」理论体系的核心文档仓库，收录从宇宙统一场方程到文明工程路线图的全套理论研究文档。

## 核心理论文档

| 文档 | 说明 |
|------|------|
| 太极统一场方程 | 宇宙统一场的数学表达 |
| 60°意识网格 | 意识科学的六边形拓扑模型 |
| 五运六气物理学 | 传统历法与现代物理的融合 |
| 实验验证路线图 | 可验证的实验设计 |
| 文明工程路线图 | 从理论到应用的落地路径 |

## 特色研究

- **384 爻能量分布与极值共振分析** — 卦象能量的数值化研究
- **太极 S 场线耦合动力学模型** — 场论的动力学表达
- **4860 年周期模型** — 天文学基础与考古学锚点
- **巳亥银河能量主轴** — 罗盘 64 卦周天校准
- **太阳系-天狼星系能量耦合模型** — 跨星系能量研究
- **太极场超空间通信模型** — 超光速诠释与通信理论
